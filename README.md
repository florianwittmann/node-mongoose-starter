# Node-Mongoose-Starter

**This project is still heavily WIP!**

Plays best with [react-redux-api-starter](https://github.com/florianwittmann/react-redux-api-starter).

## installation

- `yarn`
- Copy `config/dev.js.example` to `config/dev.js` and fill in valid values for the variables (see below)

## Environment variables

For production use set environment variables like specified in `config/prod.js`

##  run local

- `yarn dev`
